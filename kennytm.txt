# Accents are applied from top to bottom. This section should include most of
# the characters from the blocks:

# - Latin-1 Supplement
# - Latin Extended-A
# - Latin Extended-B
# - Latin Extended Additional
# - Latin Extended-C (some)

# Symbols appear on the left:
#
# ` → ` (grave)
# ' → ´ (acute)
# ^ → ˆ (circumflex)
# ~ → ˜ (tilde)
# " → ¨ (diaeresis / umlaut)
# o → ˚ (ring)
# , → ¸ (cedilla)
# , → ˛ (ogonek)
# - → ¯ (macron)

# . → ˙ (dot)
# / → (slash or strike-through)
# '' → ˝ (double acute)
# `` → ˵ (double grave)
# v → ˇ (caron)
# U → (inverted breve)
# ; → (cedilla/comma) (used when , already means another thing)
# ? → (hook)

# Symbols appear on the right:
#
# ' → horn or hook (for vietnamese)
